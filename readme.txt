The best live reload plugin for theme development.

== Description ==

The plugin will check every second that your active Wordpress theme has changed.
If it has changed, it will reload your browser.





== Installation ==

Upload the Theme_LiveReload plugin to your blog, activate it.

The plugin is only active when debug mode is enabled.
You can enable debug mode in `wp-config.php` with `WP_DEBUG` constant value.

1, 2, 3: You're done!

You can also define additional allowed IP addresses in your Live Reload settings section.
